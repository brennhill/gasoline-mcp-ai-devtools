## Goal

Turn product feedback into clear, replayable evidence.

## Week 1 Path

1. Review [Discover Workflows](/discover-workflows/).
2. Learn [Product Demos](/guides/product-demos/).
3. Build communication-ready recordings with [Demo Scripts](/guides/demo-scripts/).
4. Capture UI feedback with [Annotation + Skills + Terminal](/guides/annotation-skill-terminal-workflow/).

## Collaboration Loop

1. Annotate exact UI context.
2. Attach logs/network traces.
3. Share replayable artifacts for Engineering and QA.
