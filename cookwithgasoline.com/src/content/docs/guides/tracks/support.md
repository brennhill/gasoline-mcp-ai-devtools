## Goal

Cut time-to-reproduce for customer-reported issues.

## Week 1 Path

1. Start with [Troubleshooting](/troubleshooting/).
2. Learn evidence capture with [Debug Web Apps](/guides/debug-webapps/).
3. Practice handoffs with [How to Share Reproducible Bug Evidence](/articles/how-to-share-reproducible-bug-evidence-with-your-team/).
4. Use [WebSocket Debugging](/guides/websocket-debugging/) for real-time support cases.

## Handoff Checklist

1. Error message + timestamps.
2. Network and WebSocket evidence.
3. Steps to reproduce.
4. Suggested owner and severity.
