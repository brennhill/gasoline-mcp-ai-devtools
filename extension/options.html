Configure your Gasoline extension

Server Connection
Configure the Gasoline server URL. Change the port if you're running the server on a different port.
Default: http://localhost:7890. Use --port flag when starting the server to use a different port.
Domain Filters
Enter domain patterns to capture logs from, one per line. Leave empty to capture from all domains.
Use * as a wildcard. For example, *.localhost matches all localhost subdomains.

Examples

  • localhost:* - All localhost ports
  • *.myapp.com - All subdomains of myapp.com
  • 127.0.0.1:3000 - Specific host and port
  • *.local - All .local domains
