Smoke Test 10 — Quick Reference

screen_recording_start · screen_recording_stop · watermark persistence
1
Ensure Kaboom is tracking this tab. Click the extension icon if needed.
2
Start tab recording (video only):
interact(screen_recording_start, name:"smoke-video-test")
3
Wait 5 seconds. The Kaboom watermark #kaboom-recording-watermark should appear in the corner.
4
Stop and verify: interact(screen_recording_stop) then observe(saved_videos, last_n:5)
5
For audio test: interact(screen_recording_start, name:"smoke-audio-test", audio:"tab"). Start YouTube playback first, then verify has_audio:true in saved_videos.
6
Watermark persistence test: start recording, then interact(refresh) — watermark should reappear after the page reloads (tests tabs.onUpdated re-send).
