# Feature Documentation

This folder contains documentation for individual features. Each feature should have:
- A product spec (user-facing description, requirements, deprecations)
- A technical spec (implementation details, code mapping)
- Any related ADRs (architecture decisions specific to this feature)

For cross-product/core documentation, see `/docs/core/`.
