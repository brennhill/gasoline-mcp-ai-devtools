# Gasoline MCP Roadmap

This document outlines the planned features and architectural improvements for Gasoline MCP.

## Current Focus (v5.3)
- Performance optimizations for high-traffic WebSocket monitoring.
- Improved selector generation for complex SPAs.
- Stabilization of the Playwright test generation engine.

## Future Plans (v6.0+)
- **Semantic DOM Snapshots**: Reduction in token usage by providing agent-curated page views.
- **Visual Grounding**: Coordinates for all interactive elements to support multi-modal agents.
- **Advanced Telemetry**: Immediate feedback loops for background errors.
